# Project Overview

## Name

CryptoLens AI

## Description

CryptoLens AI is a dark-themed cryptocurrency dashboard with educational AI features. The application allows users to explore real cryptocurrency market data and request simple AI explanations about crypto assets and Web3 concepts.

## Target User

People who are starting to learn about cryptocurrencies, blockchain and Web3.

## Main Features

- Cryptocurrency dashboard
- Search and filter cryptocurrencies
- Cryptocurrency detail page
- Basic price chart
- AI explanation for selected cryptocurrencies
- Educational Web3 concepts page
- Responsive dark UI

## Out of Scope

- Backend
- Database
- Authentication
- Real wallet integration
- Real transactions
- Financial advice
- Portfolio tracking with real money

## Purpose

This project is designed as a learning and portfolio project.
