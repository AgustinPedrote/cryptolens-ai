# CryptoLens AI

CryptoLens AI is a web application built with React, TypeScript and Tailwind CSS that allows users to explore cryptocurrency market data and generate basic AI-powered educational explanations about crypto assets and Web3 concepts.

## Project Goals

- Build a portfolio-ready frontend project.
- Practice React with TypeScript.
- Use Tailwind CSS for a modern dark dashboard UI.
- Consume real cryptocurrency data from an external API.
- Integrate a basic AI assistant.
- Add testing with Vitest and React Testing Library.
- Dockerize the frontend application.
- Deploy the app to Netlify.
- Work with AI tools using a structured workflow.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Axios
- TanStack Query
- Recharts
- Gemini API
- Vitest
- React Testing Library
- Docker
- Netlify
